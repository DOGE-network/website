## ğŸ”§ How to Edit Different Content Types

### Editing Meeting Information

1. **Go to `/admin`** and click "Edit Meetings"
2. **Find the meeting** you want to update
3. **Click "Edit"** next to the meeting
4. **Update the fields:**
   - **Title:** Meeting name
   - **Date:** Meeting date (YYYY-MM-DD format)
   - **Duration:** How long the meeting lasted
   - **Description:** Brief summary of the meeting
   - **YouTube Video ID:** Just the ID part (e.g., "dQw4w9WgXcQ")
5. **Click "Save Changes"**

### Editing Event Schedules

1. **Go to `/admin`** and click "Edit Schedule"
2. **Find the event** you want to update
3. **Click "Edit"** next to the event
4. **Update the fields:**
   - **Event Name:** Name of the event
   - **Date:** Event date (YYYY-MM-DD format)
   - **Location:** Where the event is held
   - **Description:** What the event is about
5. **Click "Save Changes"**

### Adding New Content

1. **Go to `/admin`** and choose the content type
2. **Click "Add New"** button
3. **Fill in all required fields**
4. **Click "Save"** to create the new content

### Writing Style
- **Keep it professional** but conversational

- **Use bullet points** for lists when possible

### Formatting Tips
- **Use bold text** for important information
- **Use headings** to organize content
- **Include links** to related pages when helpful
- **Add emojis** to make content more engaging

### Meeting Notes Best Practices
- **Capture key decisions** clearly
- **List action items** with assigned names and due dates
- **Include next steps** for follow-up
- **Note any important resources** mentioned

## ğŸ¯ Common Editing Tasks

### Updating a Meeting Recording

1. **Get the new YouTube video ID** from the video URL
2. **Go to the meetings page** and click "Edit Meetings"
3. **Find the meeting** and click "Edit"
4. **Replace the old video ID** with the new one
5. **Update the meeting date** if needed
6. **Click "Save Changes"**

### Adding a New Meeting

1. **Go to `/admin`** and click "Edit Meetings"
2. **Click "Add New Meeting"**
3. **Fill in the meeting details:**
   - Title: "Monthly Check-in - [Month] [Year]"
   - Date: [YYYY-MM-DD]
   - Duration: [X] minutes
   - Description: Brief summary of what was discussed
   - Video ID: YouTube video ID (if recorded)
4. **Click "Save"**

### Updating Event Information

1. **Go to `/admin`** and click "Edit Schedule"
2. **Find the event** and click "Edit"
3. **Update the relevant fields:**
   - Change date if rescheduled
   - Update location if changed
   - Modify description if details changed
4. **Click "Save Changes"**

## ğŸ” Troubleshooting

### Content Not Updating
- **Check if you saved** - look for a success message
- **Refresh the page** to see your changes
- **Wait a few minutes** - changes may take time to appear
- **Contact technical support** if issues persist

### Can't Find What You're Looking For
- **Use the search function** in the admin interface
- **Check different content categories** - content might be in a different section
- **Look at the website navigation** to understand the structure
- **Ask a team member** for help locating content

### Formatting Issues
- **Check your markdown syntax** - use the preview function
- **Avoid complex formatting** - stick to basic markdown
- **Use the templates** provided for consistent formatting
- **Test your changes** in the preview before saving
